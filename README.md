# 🛋️ A&C Renova Estofados - Landing Page

Uma Landing Page responsiva e focada em conversão desenvolvida para um negócio local de higienização de estofados. O principal objetivo do projeto é atrair clientes da região de São Paulo (Capital e Grande SP) e direcioná-los para um orçamento rápido via WhatsApp.

🔗 **[Visualizar o projeto online](https://juliana15paak.github.io/ac-renova-estofados/)**

## 🚀 Principais Funcionalidades e Soluções

- **Foco em Conversão (UX/UI):** Design direto ao ponto, com chamadas para ação (CTAs) claras direcionando para o WhatsApp com mensagens pré-configuradas.
- **SEO Local:** Estrutura de meta tags e conteúdo otimizado para motores de busca, focando em palavras-chave geográficas (Bairros da ZN, ZL e Grande SP) para ranqueamento regional.

## 🛠️ Tecnologias Utilizadas

- **Design & Prototipagem:** Figma
- **Front-end:** HTML5 e CSS3
- **Deploy/Hospedagem:** GitHub Pages
- **Integração:** API do WhatsApp (Links parametrizados)

## 🧠 Aprendizados do Projeto

Durante o desenvolvimento deste projeto, pude aprimorar a transição de um layout Desktop-First desenhado no Figma para uma aplicação web totalmente responsiva. Também apliquei conceitos práticos de SEO voltado para negócios locais, entendendo como a semântica do código impacta diretamente na visibilidade de um serviço na internet.
